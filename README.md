Paradise Nursery is a React-based shopping application that allows users to browse
houseplants, add them to a cart, update quantities, and view total cost before checkout.

## Features
- Landing page with background image
- Product listing page
- Shopping cart with quantity controls
- Redux Toolkit for state management

## Technologies Used
- React
- Redux Toolkit
- React Router
- CSS
